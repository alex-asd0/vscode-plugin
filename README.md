# Time tracker
это плагин для отслеживания времени работы в vscode

авторство Маренин Алексей Сергеевич M3101

## ТЗ (возможности)
1)	При запуске vscode запускается таймер в статус баре
2)	При неактивности в течение 20 секунд таймер останавливается
3)	При возобновлении активности (нажатие клавиш) таймер снова запускается
4)	При закрытии окна vscode значение таймера сохраняется как общее время работы
5)	При открытии окна таймер обнуляется, но данные общего времени работы сохраняются
6)	При выполнении команды Time Tracker: showStatistic можно посмотреть статистику – общее время работы в этом workspace
